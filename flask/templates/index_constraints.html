Index Sequence Constraints

Index Sequence Constraints are built from Allen's interval algebra to describe locational constraints between the intervals of biopolymers. Index Sequence Constraints do not describe constraints with biopolymer strandedness in mind. To describe constraints which work with stranded components see Topological Sequence Constraints

A equals B

This constrains that A and B contain the same start and end positions.

A contains B

The 'contains' constraint aswell as the 'within' constraint are the supersets of the 'equals', 'ends_with' and 'starts_with' constraints. This constrains that B must have a start point >= to the start point of A and have an end point <= to the end point of A .

A within B

This constrains that A must have a start point >= to the start point of B and an end point <= to the end point of B.

A ends_with B

This constrains that the end points of A and B are equal and that the start point of B > than the start point of A.

A starts_with B

This constrains that the start point of A and B are equal and that the end point of B is < than the end point of A.

A touches B

This constrains that the end point of A is touching the start point of B. The start point of B is equal to the end point of A + 1

A gap_then B

This constrains that the both the start and end point of A are < than then start and end point of B. The start point of B is equal to the end point of A + i where i = 2,3,4,...,n.

A then B

The 'then' constraint is the superset of 'gap_then' and 'touches'. It describes that A comes before B but does not specify if there is a gap or if they are touching.

A overlaps B

This constrains that the end of A overlaps with the start of B. The start point of A is < than the start point of B. The end point of A is >= to the start point of B and is < than the end point of B.
